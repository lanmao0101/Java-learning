

# Java学习项目

这是一个Java基础学习项目，包含了从入门到进阶的多个练习示例。

## 项目结构

项目按天数组织，每天包含不同的Java基础知识点练习：

- **day01**: Java入门 - 第一个Java程序
- **day02**: 基础语法和运算符练习
- **day03**: 流程控制语句（选择结构）
- **day04**: 循环结构和数组练习
- **day05**: 方法和数组的高级应用
- **day06**: 内存分析练习
- **day07**: 综合应用案例
- **day08**: 面向对象基础 - 类和对象
- **day09**: 面向对象特性 - 封装与构造方法
- **day10**: 静态成员、final关键字与枚举
- **day11**: 继承与多态

## 环境要求
- JDK 8.0 或更高版本
- 任何支持Java的IDE（如IntelliJ IDEA、Eclipse）

## 运行方式

使用命令行编译运行：
```bash
javac src/com/day01/holleword.java
java -cp src com.day01.holleword
```

或使用IDE直接运行对应的main方法。

## 适用人群
适合Java初学者，通过实践掌握Java基础语法和编程思想。

## 学习建议
1. 按天数顺序学习
2. 每个demo都包含`main`方法，可直接运行
3. 建议先理解代码逻辑，再动手修改尝试
4. 多做练习，加深记忆
