# Java Learning Project

This is a foundational Java learning project containing multiple practice examples ranging from beginner to advanced levels.

## Project Structure

The project is organized by day, with each day covering different fundamental Java topics:

- **day01**: Java Introduction - First Java Program
- **day02**: Basic Syntax and Operators Practice
- **day03**: Control Flow Statements (Selection Structures)
- **day04**: Loop Structures and Array Practice
- **day05**: Advanced Applications of Methods and Arrays

## Main Content

### Day01 - Java Introduction
- Understanding basic program structure

### Day02 - Basic Practice
- BMI Calculator
- Variable and constant usage
- Operator exercises
- WeChat feature simulation

### Day03 - Control Flow
- Sequential structure
- if-else statements
- switch-case statements
- Logical operator applications

### Day04 - Loops and Arrays
- for loops
- while loops
- Array operations
- Comprehensive applications

### Day05 - Methods and Arrays
- Method definition and invocation
- Array statistics (sum, average, maximum, minimum, etc.)
- Implementation of comprehensive case studies

## Environment Requirements
- JDK 8.0 or higher
- Any Java-compatible IDE (e.g., IntelliJ IDEA, Eclipse)

## How to Run

Compile and run using the command line:
```bash
javac src/com/day01/holleword.java
java -cp src com.day01.holleword
```

## Target Audience
Suitable for Java beginners seeking to master fundamental syntax and programming concepts through hands-on practice.

## Learning Suggestions
1. Follow the days in sequence.
2. Each demo includes a `main` method and can be executed directly.
3. First understand the code logic before modifying and experimenting.
4. Practice frequently to reinforce memory.
